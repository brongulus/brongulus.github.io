Creating a blog using ox-hugo, org mode and github pages

  1. Install hugo from your package manager.
  2. Create a new site:
hugo new site blog
  1. Add a theme:
cd blog
git init
git submodule add <theme_url> themes/<name>
  1. Install ox-hugo in emacs
;; goes in packages.el
(package! ox-hugo)

;; goes in config.el
(use-package ox-hugo
  :after ox)
